Verification Set-up Guide

  1. Configure **SMTP Gmail Address** and **App Password** to send codes to patients.
  2. If these values are left empty, the app runs in *Mock Sandbox mode*.
  3. In Mock mode, authentication codes will NOT be sent to actual emails. Instead, they write to a local log file.
  4. Administrators can fetch these codes dynamically from the **Admin Panel -> Sandbox Verification Logs** list to register/log in.
